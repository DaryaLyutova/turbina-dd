## Инструкция по установке:

1. Клонировать репозиторий:

```
git clone https://github.com/dianadomino24/turbina-dd.git
```

2. Установить зависимости:

```
npm install
```

3. Доступные сценарии:

```
    * режим разработки:

      npm start

    * создание production версии приложения:

      npm build
```
